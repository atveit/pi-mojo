# Real-Time AI Event Streaming Example (example_event_stream.mojo)

This example demonstrates how to establish a live streaming connection with a cloud LLM (Google Gemini API) and process raw Server-Sent Events (SSE) data chunks in real-time. It showcases Mojo's native FFI capabilities to read streaming HTTP connections and print tokens immediately as they are generated by the model.

### ⚙️ How it Works
1. **Streaming Endpoint**: The agent targets the `/streamGenerateContent` endpoint of the Gemini API.
2. **Buffer Reading**: Utilizing Python's `urllib.request` libraries, the agent reads the response line-by-line asynchronously in a loop.
3. **Ultra-Fast Parsing**: An optimized string-matching pattern scans each raw text chunk for the `"text"` field and extracts it without the overhead of complete JSON tree parsing.
4. **Token Flushing**: Each token is outputted immediately using a flushed stdout write stream, giving a real-time word-by-word visual effect in the terminal.

### 🔍 Code Explanation & Key Snippets

This example demonstrates how to implement low-overhead, real-time event streaming in Mojo:

#### 1. FFI Streaming Setup
Utilizing Python's `urllib.request` connection objects to fetch streaming HTTP content incrementally:
```mojo
var url = "https://generativelanguage.googleapis.com/v1beta/models/gemini-3.5-flash:streamGenerateContent?key=" + api_key
var response = urllib.urlopen(req, timeout=15)
```

#### 2. Event Consumption Loop
Reading line-by-line bytes incrementally from the socket stream without waiting for the full connection payload to close:
```mojo
while True:
    var line_bytes = response.readline()
    if not line_bytes:
        break
    var line = line_bytes.decode("utf-8")
```

#### 3. Optimized String Token Extraction
Using dynamic string searches (`find`/`rfind`) to isolate token values, completely bypassing heavy JSON parser tree building:
```mojo
var target = String('"text": "')
var idx = Int(py=py_line.find(target))
if idx != -1:
    var start_pos = idx + 9
    var end_pos = Int(py=py_line.rfind('"'))
    if end_pos > start_pos:
        var token = String(py_line[start_pos:end_pos])
        var clean_token = token.replace("\\n", "\n").replace("\\t", "\t").replace('\\"', '"')
        _ = sys.stdout.write(clean_token)
        _ = sys.stdout.flush()
```
